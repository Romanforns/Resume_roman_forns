Risk and credit-score engine for DeFi wallets (Aave V3 on Base). On-chain indexer subscribing via WebSocket to Pool events, normalised to a canonical USD model and persisted idempotently in PostgreSQL. Risk feature engine and configurable weighted scoring. REST API (Express/TypeScript) for score, features and recompute by wallet address. Next.js frontend for address lookup and score/features visualisation. Docker deployment.
